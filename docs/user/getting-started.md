# Getting Started with Reelspot

Welcome to Reelspot! This guide will help you get started with the minimal, premium video downloader platform.

## What is Reelspot?

Reelspot is a modern, high-performance web application designed to help you download videos and reels from your favorite social media platforms like Instagram, YouTube, Facebook, and TikTok. It features a clean, ad-free interface and handles downloads locally or via API integrations.

## Prerequisites

- A modern web browser (Edge, Chrome, Firefox, Safari)
- An active internet connection

## Accessing the Application

Reelspot is a web-based application. Simply navigate to the deployed URL (or `http://localhost:5173` if running locally) to access the dashboard.

## Dashboard Overview

Upon launching Reelspot, you will be greeted by the **Dashboard**.

- **Welcome Section**: Personalized greeting and quick stats.
- **Quick Download**: Paste a video URL to start downloading immediately.
- **System Health**: View the status of your connected API platforms.
- **Recent Activity**: Quickly access your latest downloads.

## How to Download a Video

1.  **Copy the Link**: Go to the social media post (Instagram Reel, YouTube Video, etc.) and copy the share link.
2.  **Paste & Download**:
    - Return to Reelspot.
    - Paste the link into the main input bar on the Dashboard.
    - Click the **Download** button (or press Enter).
3.  **Preview & Save**:
    - The video metadata will resolve.
    - Click **Download Video** (or Audio/Image options if available).
    - The file will be saved to your device, and the record added to your **History**.

## Managing History

Navigate to the **History** page via the sidebar to:

- View all your past downloads.
- Filter by platform (Instagram, YouTube, etc.) or type (Video, Reel).
- Re-download items without fetching metadata again.
- Delete items to keep your history clean.

## Settings

Use the **Settings** page to:

- Configure **API Keys** for different platforms.
- Adjust **Preferences** like auto-download, notifications, and theme.
- Manage your **Profile** and **Data** (Export CSV/JSON).
