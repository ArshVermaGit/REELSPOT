# Security Policy

## Supported Versions

We are currently only supporting security updates for the latest stable version of Reelspot.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you've discovered a security vulnerability in Reelspot, please report it privately.

**Do not open a public issue.** This allows us to fix the vulnerability before it's public knowledge.

To report a vulnerability, please email Arsh Verma at [arsh.v.creates@gmail.com](mailto:arsh.v.creates@gmail.com).

Please include the following information in your report:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact of the vulnerability.
- Any suggested fixes or mitigations.

We will acknowledge your report within 48 hours and provide a timeline for a fix. We appreciate your responsible disclosure.
