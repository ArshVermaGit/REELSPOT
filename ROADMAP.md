# Roadmap

This document outlines the planned features and milestones for Reelspot.

## Q1 2026: Foundation & Polish

- [x] Initial Release with core platforms.
- [x] Hyper-premium UI and animations.
- [x] Persistent history with local storage.
- [/] Comprehensive Documentation (GitHub Perfection).

## Q2 2026: Expansion & Features

- [ ] **Additional Platforms**: Add support for Twitter (X), Pinterest, and LinkedIn.
- [ ] **Batch Downloads**: Allow users to download multiple videos at once via a list of URLs.
- [ ] **Quality Selection**: Choose between different video resolutions (360p, 720p, 1080p, 4K).
- [ ] **Cloud Sync**: Optional Supabase integration for syncing history across devices.

## Q3 2026: Ecosystem & Community

- [ ] **Reelspot Desktop App**: Build a cross-platform desktop version using Electron.
- [ ] **Browser Extension**: Quick download button directly on social media sites.
- [ ] **Public API**: Stable API for third-party developers to integrate Reelspot features.

## Q4 2026: Advanced Tools

- [ ] **In-App Video Trimming**: Simple tools to trim videos before downloading.
- [ ] **Audio Extraction**: Option to download only the audio (MP3) from videos.
- [ ] **Analytics Dashboard**: For pro users to track their usage and platform trends.

---

> [!NOTE]  
> This roadmap is a living document and is subject to change based on community feedback and technological shifts.
